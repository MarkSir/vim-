

该项目是vim编辑器的配置文件，依赖nerdtree。

请先下载nerdtree


之后将该项目clone到本地，并将文件复制到`~`目录下

```
cd ~

git clone https://github.com/shanezh/vim-.git

cp vim-/* ~/

```



### 快捷键介绍

进入vim模式

输入`control + n`会 打开/关闭 树形目录

输入 `o`是打开文件

输入 两次`tab` 切换窗口

输入 `t` 会将文件在新的 列窗口下打开

输入 `gt` 会切换列窗口

输入 `数字 + gt` 会切换到对应的列窗口
